Proyecto en Python: Piedra, Papel o Tijeras - Versión Mejorada 'V2.0'

Este repositorio contiene una versión mejorada del clásico juego Piedra, Papel o Tijeras, desarrollado en Python como parte de mi proceso de 
aprendizaje y práctica. A diferencia de la versión anterior, esta incluye funcionalidades que enriquecen la experiencia del usuario:


Características principales:

Ingreso de nombre personalizado: El juego solicita al usuario su nombre antes de comenzar.

Registro de partidas: Se crea un directorio específico para almacenar las partidas de cada jugador.

Sistema de puntuación: Se lleva un marcador que registra los puntos obtenidos en cada ronda.

Control de flujo de juego: El usuario puede decidir si desea continuar o finalizar la partida después de cada ronda.


El objetivo de este proyecto está diseñado para ser útil, entretenido y motivador para quienes están comenzando en el mundo del desarrollo. 
Mi intención es compartir código claro, funcional y fácil de entender, que sirva como base para futuras mejoras o personalizaciones.
